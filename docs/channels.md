# Каналы

Каналы в Go — это типизированные конвейеры, через которые вы можете отправлять и получать значения с помощью операторов канала <-. Они обеспечивают синхронизацию между горутинами, гарантируя, что отправленное значение будет получено, когда приемник готов, и наоборот.

Рассмотрим два основных типа каналов, которые могут использоваться:

1. **Небуферизованный канал (Unbuffered Channel):** Такой канал имеет нулевую емкость. Отправка в небуферизованный канал блокируется до тех пор, пока другая горутина не будет готова принять значение. Прием из небуферизованного канала блокируется до тех пор, пока другая горутина не отправит значение. Это обеспечивает строгую синхронизацию.

2. **Буферизованный канал (Buffered Channel):** Такой канал имеет заданную емкость. Отправка в буферизованный канал блокируется только тогда, когда буфер заполнен. Прием из буферизованного канала блокируется только тогда, когда буфер пуст.
